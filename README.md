# agegate

Simple jQuery popup that requires a user to type their birthday into a form with optional remember me cookie.

Set

```javascript
age: '21',
cookieLength: '7',
cookieName: 'ageGate'
```

Call

```javascript
jQuery.ageGate();
```

or pass settings

```javascript
jQuery.ageGate({
  age: '21',
  cookieLength: '7',
  cookieName: 'ageGate'
});
```
